# Installation

Get started with Blok components in your React project.

## Requirements

- Node.js 18.0 or later
- React 18.0 or later
- Tailwind CSS 3.0 or later

## CLI Installation

The easiest way to add components to your project is using the Blok CLI.

<Steps>

### Install dependencies

First, make sure you have the required dependencies:

```bash
npm install react react-dom
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

### Configure Tailwind CSS

Add the following to your `tailwind.config.js`:

```js title="tailwind.config.js" {3-6}
/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    "./app/**/*.{js,ts,jsx,tsx,mdx}",
    "./components/**/*.{js,ts,jsx,tsx,mdx}",
  ],
  theme: {
    extend: {},
  },
  plugins: [],
}
```

### Initialize the CLI

Run the init command to set up your project:

```bash
npx blok-ui@latest init
```

This will:
- Create a `components.json` file
- Set up your import aliases
- Configure your project structure

### Add a component

Now you can add any component:

```bash
npx blok-ui@latest add button
```

</Steps>

## Manual Installation

You can also manually copy component code into your project.

### Install dependencies

Install the required dependencies for the components you want to use:

```bash
npm install @radix-ui/react-slot class-variance-authority clsx tailwind-merge
```

### Add utility functions

Create a `lib/utils.ts` file:

```ts title="lib/utils.ts"
import { clsx, type ClassValue } from "clsx"
import { twMerge } from "tailwind-merge"

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs))
}
```

### Copy component code

Copy the component code from the documentation page and paste it into your project.

<Callout type="info">
  Each component page includes installation instructions and the full source code.
</Callout>
